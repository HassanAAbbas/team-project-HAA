Hassan’s Store
This project is my individual store module for the Full Stack Application Team Project. It includes a complete CRUD API for product data, a MongoDB Atlas connection, a simple frontend, and an automated test for the GetAll endpoint.

Features
- Product schema with: storeId, storeName, productId, productName, price  
- Full CRUD routes (GET all, GET one, POST, PUT, DELETE)  
- MongoDB Atlas integration using Mongoose  
- Simple frontend that loads and displays products  
- Individual GetAll test using Supertest  
- automate.js script prepared to run my test and a placeholder for teammate test  

Purpose
This module is part of a team integration project where each student builds their own store and later integrates all stores into a shared GetAll endpoint.
